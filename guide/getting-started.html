# Install

Note

Globally installed VuePress won't be able to find locally installed plugins.

yarn install -D vuepress-plugin-catalog-graph

# Enalbe in VuePress

module.exports = {
  plugins: [ 'vuepress-plugin-catalog-graph' ]
}
