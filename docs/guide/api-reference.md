# API Reference

## GraphNode

Each GraphNode represents an actual node in the Catalog Graph.

### node.title

* Type: `String`

### node.regularPath

* Type: `String`

### node.id

* Type: `String`

### node.frontmatter

* Type: `Object`

### node.parent

* Type: `{Object} GraphNode`

### node.children

* Type: `Array<GraphNode>`
